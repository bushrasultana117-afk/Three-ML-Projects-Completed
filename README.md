# 🩺 Multiple Disease Prediction System

An AI-powered healthcare application built with **Python, Machine Learning, and Streamlit** to predict the likelihood of multiple diseases from user input.

## 🚀 Features

* ❤️ Heart Disease Prediction
* 🧠 Parkinson's Disease Prediction
* 🩸 Diabetes Prediction
* Simple & Interactive Streamlit Interface

## 🤖 Machine Learning Models

| Disease                | Algorithm                       | Accuracy |
| ---------------------- | ------------------------------- | -------- |
| ❤️ Heart Disease       | Random Forest Classifier        | **100%** |
| 🧠 Parkinson's Disease | Support Vector Classifier (SVC) | **88%**  |
| 🩸 Diabetes            | Gaussian Naive Bayes            | **77%**  |

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* NumPy
* Pandas
* Pickle
